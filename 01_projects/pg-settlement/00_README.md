# PG Settlement Project

PG/정산/대사 업무를 프로젝트 단위로 기록하는 공간입니다.

이 폴더는 도메인 지식만 쌓는 곳이 아니라, 현재 업무 기록을 포트폴리오 소재, 자동화 아이디어, 데이터 분석 질문으로 연결하는 프로젝트 허브입니다.
