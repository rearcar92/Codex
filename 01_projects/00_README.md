# Projects

현재 업무와 개인 프로젝트를 기록하는 중심 공간입니다.

프로젝트는 최종 결과물만 모으는 곳이 아니라, 업무 기록에서 포트폴리오 소재, 자동화 아이디어, 데이터 분석 질문, AI 서비스 아이디어를 뽑아내는 출발점입니다.

## 시작 문서

- [[MOC - 프로젝트]]
- [[MOC - 현재 업무]]

## 하위 구조

- `current-work`: 현재 업무 기록
- `pg-settlement`: PG/정산/대사 프로젝트

## 운영 원칙

- 새 업무 기록은 `current-work`에서 시작합니다.
- 반복되거나 의미 있는 업무는 개별 프로젝트로 묶습니다.
- 프로젝트마다 포트폴리오, 자동화, 데이터 분석으로 이어지는 링크를 남깁니다.
