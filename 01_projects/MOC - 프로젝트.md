# MOC - 프로젝트

현재 하고 있는 일과 앞으로 만들 프로젝트를 관리하는 중심 지도입니다.

프로젝트는 단순 폴더가 아니라, 업무 기록, 포트폴리오 소재, 자동화 아이디어, 데이터 분석 질문, AI 서비스 아이디어가 파생되는 출발점입니다.

## 프로젝트 운영 방식

각 프로젝트는 아래 흐름으로 기록합니다.

```text
프로젝트 기록
→ 업무 맥락
→ 문제/목표
→ 산출물
→ 포트폴리오 소재
→ 자동화 아이디어
→ 데이터 분석 질문
→ AI 서비스 아이디어
```
