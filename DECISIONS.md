# Decisions

## 2026-07-16: Obsidian + Git + Google Drive 3분할 운영

### 결정

Codex 프로젝트 관리는 Obsidian, GitHub, Google Drive를 역할별로 나눈다.

### 이유

- Obsidian은 Markdown 기반이라 지식, 기획 초안, 연결형 메모 관리에 적합하다.
- GitHub는 변경 이력, 여러 PC 동기화, 복원, 브랜치 관리에 적합하다.
- Google Drive는 PDF, 이미지, 엑셀, 발표자료 등 무거운 산출물 보관에 적합하다.

### 운영 기준

- Obsidian: 생각, 지식, 기획 맥락
- GitHub: 정리된 문서, 코드, 자동화 스크립트, 운영 기준
- Google Drive: 최종 산출물, 공유 파일, 큰 첨부자료

### 보류

- GitHub 원격 저장소 이름과 공개 범위
- Obsidian Sync 사용 여부
- Drive 폴더 공유 범위
